# Allerton Folk School Event Registration Analysis

This repository presents key findings from the Allerton Folk School event registration analysis. The project uses cleaned registration, geography, pricing, retention, and survey data to summarize program performance and support future decisions related to outreach, programming, pricing, and participant engagement.

## Overview

The analysis focuses on several core questions:

- Who is attending Allerton Folk School programs?
- How many participants return for additional classes?
- How does demand vary across price bands?
- Where are participants coming from geographically?
- What do post-course survey responses suggest about participant experience?

The KPI tables below provide a concise summary of the major findings.

## Purpose

These outputs are intended to provide Allerton with a reusable reporting structure for monitoring program performance over time. Future CourseStorm registration exports and survey data can be incorporated into the same framework to support ongoing analysis and decision-making.
